## ⚡️ Installation

1. Clone the repo
   ```sh
   git clone https://github.com/OzerOzturk/ciceksepeti-ecommerce.git
   ```
2. Navigate to the project folder

   ```sh
   cd ciceksepeti-ecommerce
   ```

3. Install dependencies
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm start
   ```
